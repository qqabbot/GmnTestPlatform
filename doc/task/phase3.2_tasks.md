# Phase 3.2 Frontend Optimization Tasks

## 1. 变量使用体验优化 (Variable Usage Experience)
- ✅ **$ Trigger Autocomplete**
  - ✅ Monitor input fields for `$` character.
  - ✅ Trigger a popup/dropdown menu when `$` is typed.
  - ✅ Fetch and display available Global and Environment variables in the list.
  - ✅ Support keyboard navigation (up/down/enter) and click selection.

## 2. 断言与提取器优化 (Assertions & Extractors)
- ✅ **Card-Style UI**
  - ✅ Convert list items to distinct "cards" instead of simple rows.
  - ✅ Improve visual hierarchy and spacing.
- ✅ **Visual Management**
  - ✅ Add prominent "Add" buttons.
  - ✅ Add visual "Delete" buttons (e.g., trash icon on card).
  - ✅ Support multi-item management (visual separation).

## 3. Body 编辑器优化 (Body Editor)
- ✅ **Monaco Editor Integration**
  - ✅ Replace existing textarea/simple editor with Monaco Editor.
  - ✅ **Dark Theme**: Default to a generic dark theme or custom dark styling.
  - ✅ **Variable Highlighting**: Syntax highlighting for `${variable}` patterns.
  - ✅ **Auto-Formatting**: JSON/XML auto-formatting capability.

## 4. 步骤管理优化 (Steps Management)
- ✅ **Sidebar Step List**
  - ✅ Move or duplicate step list to a dedicated left-side sidebar.
  - ✅ Display complete list of steps in the test case.
- ✅ **Drag-and-Drop Sorting**
  - ✅ Allow reordering steps via drag-and-drop in the sidebar.
- ✅ **Step Actions**
  - ✅ **Disable/Enable**: Toggle switch or button to skip steps during execution.
  - ✅ **Delete**: Button to remove steps.
